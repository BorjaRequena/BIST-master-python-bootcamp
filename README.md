# BIST-master-python-bootcamp

Contents for the scientific python bootcamp of the BIST master of Multidisciplinary Research in Experimental Sciences (2020 course).

The contents of this year's bootcamp are designed based on [previous iterations](https://github.com/germannp/BIST-Python-Bootcamp).

Suggested Installation
----------------------
1. Anaconda3 with Jupyter and Spyder (or another editor)
2. The python packages NumPy, Pandas, SciPy, Matplotlib, Seaborn & Scikit-learn
3. Git
